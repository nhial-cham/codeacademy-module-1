## Problem - Challenge 74
# Problem: Rectangle Area Calculator

## Objective
Create a function that calculates the area of a rectangle.

## Scenario
You are building a simple tool for a construction company that needs to quickly 
calculate the area of rectangular spaces (e.g., rooms, floors, or land plots).

The formula to calculate the area of a rectangle is:

    area = width * height

## Requirements
1. Create a function named `calculateArea`.
2. The function must receive:
   - `width` (number)
   - `height` (number)

3. The function should:
   - Calculate the area using the formula
   - Return the result

## Example

### Input:
width = 5  
height = 3  

### Output:
area = 15
