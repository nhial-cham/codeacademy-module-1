Problem 3

Description

Write a program that asks the user to enter a number. The program should determine whether the number is positive, negative or zero.

Input:

A number entered by the user (can be positive, negative, or zero)

Conditions:

If the number is greater than 0 → it is positive


If the number is less than 0 → it is negative


If the number is equal to 0 → it is zero


Output:


Display: "The number is positive"


Display: "The number is negative"


Display: "The number is Zero"
