# Problem: Sum of Multiples of 3

In mathematics, a **multiple of 3** is any number that can be divided by 
**3 without a remainder**.

Examples of multiples of 3:

- 3
- 6
- 9
- 12
- 15

Your task is to write a **JavaScript program using Node.js** that calculates the 
**sum of all multiples of 3 between 1 and 20**.

## Requirements

- Use a **`for` loop** to iterate from **1 to 20**.
- In each iteration, determine if the number is a **multiple of 3**.
- If the number is a multiple of 3, add it to a variable that stores the total sum.
- Print each multiple of 3 found.
- After the loop finishes, print the **total sum**.

## Expected Output

```
Multiple of 3: 3
Multiple of 3: 6
Multiple of 3: 9
Multiple of 3: 12
Multiple of 3: 15
Multiple of 3: 18
Total sum: 63
```

## Notes

- Do **not** use user input (`prompt`, `readline`, etc.).
- Define all values **directly in the code**.
- The goal of this exercise is to practice:
  - `for` loops
  - conditional statements
  - the modulus operator `%`
  - accumulation with variables
