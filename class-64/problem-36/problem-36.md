# Problem: Count Even and Odd Numbers

In mathematics, numbers can be classified as **even** or **odd**:

- **Even numbers** are divisible by 2 with no remainder.
- **Odd numbers** leave a remainder when divided by 2.

Your task is to write a **JavaScript program using Node.js** that counts how many 
**even numbers** and how many **odd numbers** exist between **1 and 20**.

## Requirements

- Use a **`for` loop** to iterate from **1 to 20**.
- In each iteration, determine if the number is **even or odd**.
- Use two variables:
  - one to count **even numbers**
  - one to count **odd numbers**
- At the end of the program, print the totals using `console.log()`.

## Expected Output

```
Even numbers: 10
Odd numbers: 10
```

## Notes

- Do **not** use user input (`prompt`, `readline`, etc.).
- Define all values directly in the code.
- The goal of this exercise is to practice:
  - `for` loops
  - conditional statements
  - counters
  - the modulus operator `%`
