# Problem: Multiplication Table Generator

A multiplication table shows the result of multiplying numbers together.

Your task is to write a **JavaScript program using Node.js** that prints the 
**multiplication tables from 1 to 5**.

Each table should show the multiplication from **1 to 10**.

## Requirements

- Use **two `for` loops**:
  - The first loop controls the **table number (1 to 5)**.
  - The second loop controls the **multiplication values (1 to 10)**.
- Print the results using `console.log()`.
- Add a blank line between each table to make the output easier to read.

## Expected Output

```
Multiplication Table of 1
1 x 1 = 1
1 x 2 = 2
...
1 x 10 = 10

Multiplication Table of 2
2 x 1 = 2
2 x 2 = 4
...
2 x 10 = 20

Multiplication Table of 3
...

Multiplication Table of 4
...

Multiplication Table of 5
...
```

## Notes

- Do **not** use user input (`prompt`, `readline`, etc.).
- Define all values **directly in the code**.
- The goal of this exercise is to practice:
  - `for` loops
  - **nested loops**
  - performing repeated mathematical operations
