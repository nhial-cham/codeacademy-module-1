# Problem: Square Numbers

In mathematics, the **square of a number** is the result of multiplying the number 
by itself.

For example:

- 2² = 4  
- 3² = 9  
- 4² = 16  

Your task is to write a **JavaScript program using Node.js** that prints the 
**square of numbers from 1 to 5**.

## Requirements

- Use a **`for` loop** to iterate from **1 to 5**.
- In each iteration, calculate the **square of the number**.
- Print the number and its square using `console.log()`.

## Expected Output

```
Number: 1 Square: 1
Number: 2 Square: 4
Number: 3 Square: 9
Number: 4 Square: 16
Number: 5 Square: 25
```

## Notes

- Do **not** use user input (`prompt`, `readline`, etc.).
- Define all values **directly in the code**.
- The goal of this exercise is to practice **`for` loop iteration and basic 
mathematical operations**.
