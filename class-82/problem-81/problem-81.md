First Example of Object-Oriented Programming (OOP) in JavaScript
Introduction
When learning Object-Oriented Programming (OOP), it is important to start with examples that represent familiar real-world objects.

A good first example should:

Be simple
Be easy to understand
Represent something students already know
Clearly demonstrate properties and methods

In this first example, we will model a Dog object.

Why a dog?

Because students already understand that dogs have:

Properties (State)
Things that describe a dog:

Name
Breed
Age
Color
Energy level

Behaviors (Methods)
Things a dog can do:

Bark
Eat
Sleep
Play

This helps students connect programming concepts with real-world thinking.
