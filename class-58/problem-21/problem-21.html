🚦 Traffic Light Simulator

Introduction

In programming, we often need to execute different actions depending on the value of a variable. JavaScript provides the switch statement to handle multiple possible cases in a clean and readable way.

Problem

You are going to simulate a simple traffic light system.

The program contains the following variable:

const light = "green";

Depending on the value of the variable, your program must display a message.

Light Color Action
red Stop the vehicle
yellow Prepare to stop
green You can go

If the color does not match any known value, the program should display:

Invalid traffic light color

Requirements

  • Use a switch statement
  • Include a default case
  • Print the result using console.log()

Your Task

Create a file called app.js and implement the logic using a switch statement.
