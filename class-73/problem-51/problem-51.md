Problem 51:
Challenge: Student Grade Analyzer
Introduction
In this challenge, you will build a Student Grade Analyzer using JavaScript 
arrays. In many real-world software systems, data is stored in collections and 
later processed to extract meaningful information. Grades in an educational system 
are a good example of this: teachers often need to analyze student scores to 
understand performance, compute averages, and determine how many students passed 
a course.

The goal of this exercise is to practice working with 
arrays and fundamental control structures in JavaScript. You will manipulate 
a list of grades and perform several operations to produce useful statistics from 
the data.

To solve this challenge, you will combine the use of:

Arrays
The length property
Basic array methods
Loops
Variables for accumulation
Conditional statements

Compared to previous exercises, this challenge introduces a small level of 
data analysis with arrays. You will iterate through a dataset, calculate 
totals and averages, compare values, and count elements that meet a specific 
condition.

Problem Description
A teacher has stored the grades of several students in the following array:

let grades = [72, 85, 90, 66, 58, 79, 91];


Using this array, write a JavaScript program that performs the following tasks.

Tasks
Print the total number of grades stored in the array.
Add a new grade 88 to the array.
Remove the last grade from the array.
Print all grades using a loop.
Calculate and print the sum of all grades.
Calculate and print the average grade.
Find and print the highest grade in the array.
Count how many students passed the course.

   
A student passes if the grade is 70 or higher.
Print the number of students who passed.
Expected Output Structure
Total grades: 7
Sum of grades: ...
Average grade: ...
Highest grade: ...
Students passed: ...


The exact numeric values will depend on the calculations performed by your program.

Goal of the Exercise
By completing this challenge, you will practice:

Iterating through arrays with loops
Performing calculations with multiple values
Tracking totals using accumulator variables
Comparing values inside a loop
Counting elements that satisfy a condition

These skills are fundamental for solving many real-world programming problems 
where collections of data must be processed and analyzed.
